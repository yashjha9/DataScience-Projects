# 📊 Capstone Project 1 – NumPy Analysis

## 📌 Overview

This project analyzes body measurement data from the NHANES dataset using NumPy.

---

## 🔍 Objectives

* Compare male and female body metrics
* Perform statistical analysis
* Visualize data using histograms and boxplots

---

## ⚙️ Implementation

* Data downloaded via Python (urllib)
* Processed using NumPy arrays
* Missing values handled
* Visualization using matplotlib

---

## 📈 Key Insights

* Male participants show higher average weight
* Greater variation observed in male data
* Ratios like waist-to-height provide better health insights than BMI

---

## 🛠️ Tech Stack

* Python
* NumPy
* Matplotlib

---

## 🚀 How to Run

1. Open Jupyter Notebook
2. Run all cells
3. Ensure internet connection for dataset download
